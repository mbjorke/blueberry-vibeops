# Customer Research

Customer discovery and validation following LOOP methodology.

## Contents

- **customer-segments.md** - Defined customer segments (VibeOps people)
- **customer-insights.md** - Interview notes, insights, quantified value propositions

## Purpose

Before building anything, we need to understand:
- Who are our customers? (segments)
- What problems do they have? (insights)
- What value can we provide? (quantified)

This is Session 0 of the Creative Platform Hackathon - the foundation for all other work.

## LOOP Alignment

- Clearly defined customer segments
- Increased understanding of customers
- Quantified customer value
- Established customer relationships
